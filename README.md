# Conference-Ticketing-System

## **Title: Conference Ticketing System**

## Description: 
The **Conference Ticketing System** is a simple real-world application designed to manage conference registrations efficiently.
This system allows users to browse conferences and register for available slots.
It tracks how many slots are open, manages delegate records, and updates availability, in real time as bookings are made or cancelled.
